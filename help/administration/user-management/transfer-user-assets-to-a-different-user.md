# 将用户资源转移给其他用户

使用“用户管理”页面的“管理报表用户和群组”可管理用户和群组，以及控制对报表、工具和报表包的访问。您可以通过Adobe Analytics中的用户管理，将一个用户帐户中的用户帐户项目（如功能板、日历事件、书签等）分配给另一个用户的帐户项目。
