# Analytics管理日志概述

日志文件有助于您了解用户登录的时间、用户使用情况、访问情况、报表包，以及管理员执行的更改。管理日志报告管理员在管理工具中执行的所有更改。此日志提供了一个由上述三个日志中的任意一个日志通往用户定义报表的途径。
