# 启用“可选时间戳”设置

可选时间戳允许我们将加盖时间戳和未加盖时间戳的数据混合到同一全局报表包中，将来自移动设备应用程序的加盖时间戳数据发送到全局报表包中，并将应用程序升级为使用离线跟踪，而无需创建新的报表包。
