# 重新运行数据馈送作业

Adobe Analytics中的数据馈送UI使数据馈送管理更加轻松。 如果您有权管理数据馈送，则可以使用数据馈送UI自行重新运行数据馈送作业，而无需客户支持团队重新发送最近的数据馈送文件。
