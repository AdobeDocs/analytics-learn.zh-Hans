# 重新运行数据馈送作业

Adobe Analytics 中的数据馈送 UI 使数据馈送管理更加容易。 如果您有权限管理数据馈送，那么您可以使用数据馈送 UI 自行重新运行数据馈送作业，而无需让客户支持团队重新发送最近的数据馈送文件。
