# “媒体播放耗时”面板

利用“媒体播放耗时”面板，媒体用户能够通过一天中所选粒度的观看时间量来了解他们的收视率。 您还可以“放大”一天中的某个时段或创建连续时段，以清楚地显示一天中各个时段的导入和导出。
