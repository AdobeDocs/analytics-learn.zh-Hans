# “媒体播放逗留时间”面板

通过“媒体播放逗留时间”面板，媒体用户可以按一天中在选定粒度内查看的时间来了解其播放量。 您还可以对一天中的某个时段进行“放大”，或创建顺序时间段以明确显示一天中各个时段的商机和商机。
