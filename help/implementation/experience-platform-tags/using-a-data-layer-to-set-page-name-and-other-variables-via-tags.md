# 使用数据层在 Experience Platform [!DNL tags] 中设置 Analytics 变量

了解如何从数据层中提取值并在Experience Platform中使用它们 [!DNL tags] 以在Adobe Analytics中填充变量。 将数据层用于 [!DNL Analytics] 而其他Experience Cloud解决方案则是最佳实践。
