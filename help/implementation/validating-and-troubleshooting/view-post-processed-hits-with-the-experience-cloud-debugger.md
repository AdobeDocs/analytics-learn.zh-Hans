# 视图后处理的Experience Cloud Debugger点击

后处理数据现在可在Experience Cloud Debugger中使用。 通过Experience Cloud登录并打开“检索后处理的点击”，查看应用VISTA和处理规则后的点击。 例如，如果您的组织严重依赖于处理规则中的上下文数据映射，您现在可以看到每个上下文数据变量直接存储在调试器中的位置。
