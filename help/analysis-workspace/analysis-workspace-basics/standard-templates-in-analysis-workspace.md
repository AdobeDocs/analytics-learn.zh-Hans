# Analysis Workspace 中的标准模板 {#standard-templates-in-analysis-workspace}

Analysis Workspace 通过[!DNL Standard Templates]（以前称为[!DNL Starter Projects]）为您的分析开一个好头。

>[!VIDEO](https://video.tv.adobe.com/v/23960/?quality=12)

[!DNL Standard Templates]是完全可自定义和可分享的模板，围绕常见的商业主题设计而成，其中具有优美、直观的可视化效果，使获得见解变得轻松许多。有关此功能的详细信息，请访问[文档](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/build-workspace-project/starter-projects.html?lang=zh-Hans)。
