# 使用 Analysis Workspace 中的 [!DNL Freeform Table Builder]

[!DNL Freeform Table Builder]可让您拖放多个维度、细分、量度和区段来生成可解答更复杂的业务问题的表格。 数据将不会即时更新；而是在您单击“生成”后更新，这样一来，表格只会更新一次，从而为您节省时间。
