# 在Analysis Workspace [!DNL Freeform Table Builder] 使用

[!DNL Freeform Table Builder] 允许您拖放多个维度、细分、指标和细分，以构建可回答更复杂业务问题的表。 数据不会即时更新；而是在您单击“Build”后更新它，从而节省了时间，因为表只更新一次。
