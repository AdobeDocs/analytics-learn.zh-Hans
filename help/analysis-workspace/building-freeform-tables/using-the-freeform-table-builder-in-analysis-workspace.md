# 在Analysis Workspace中使用[!DNL Freeform Table Builder]

[!DNL Freeform Table Builder] 允许您在许多维度、细分、量度和细分中拖放，以构建可回答更复杂业务问题的表。数据不会即时更新；相反，它将在您单击“构建”后更新，从而节省您的时间，因为表只更新一次。
