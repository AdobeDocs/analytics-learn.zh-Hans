# 在Analysis Workspace中使用[!DNL Freeform Table Builder]

[!DNL Freeform Table Builder] 允许您拖放许多维度、划分、量度和区段，以构建可回答更复杂业务问题的表。数据不会即时更新；相反，在您单击“生成”后，它将进行更新，从而节省您的时间，因为表只更新一次。
