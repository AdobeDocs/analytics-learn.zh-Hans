# [!DNL Time-parting] Analysis Workspace 中的维度 {#time-parting-dimensions-in-analysis-workspace}

[!DNL Time-parting] 将收集的点击时间戳分为更有意义的维度，如“小时（一天）”或“星期（星期）”。 这些维度现在在Analysis Workspace开箱即用。
