# [!DNL Time-parting] Analysis Workspace 中的维度 {#time-parting-dimensions-in-analysis-workspace}

[!DNL Time-parting] 会获取收集的点击的时间戳，并将其划分为更有意义的维度，例如“小时”或“星期”。这些维度现在在Analysis Workspace中是开箱即用的。
