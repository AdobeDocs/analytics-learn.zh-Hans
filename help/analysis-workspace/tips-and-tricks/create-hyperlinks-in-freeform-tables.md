# 在自由格式表中创建超链接

了解如何为一个或多个维度项目创建超链接，使其可在Analysis Workspace的自由格式表中点击。 您可以使用变量为具有URL值的维度项目、具有非URL值的维度项目的自定义URL或多个维度项目的动态自定义URL创建超链接。 有关详细文档，请访问[为自由格式表中的维度创建超链接](https://experienceleague.adobe.com/en/docs/analytics/analyze/analysis-workspace/visualizations/freeform-table/freeform-table-hyperlinks)。
