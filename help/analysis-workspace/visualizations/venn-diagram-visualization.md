# Analysis Workspace中的维恩图可视化

在Analysis Workspace中，您可以使用维恩图可视化图表以可视方式显示区段的重叠和交集。 很容易在几秒钟内绘制出令人惊叹的、有洞察力的维恩图。 此视频将向您展示如何操作。
