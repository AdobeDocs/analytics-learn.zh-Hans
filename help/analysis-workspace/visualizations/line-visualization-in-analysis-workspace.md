# Analysis Workspace 中的折线图可视化图表

折线图可视化图表描述趋势时段内的量度。可视化图表设置允许您自定义每个折线图可视化的外观，并包括“显示 X/Y 轴”、“显示最小值/最大值标签”和“显示趋势线”的设置。
