# 使用 Workspace 中的“组件”下拉菜单

“组件”下拉菜单允许您从兼容组件的列表中进行选择以构建分析。如果找不到所需的组件，也可以直接从下拉列表中创建新组件。如果您有更高级的用例，这些空间也将继续作为放置区域。
