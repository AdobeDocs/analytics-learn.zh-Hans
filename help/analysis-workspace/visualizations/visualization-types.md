# 可视化类型和概述 {#visualization-types-in-analysis-workspace}

了解可放入Analysis Workspace项目中的各种可视化图表或类型，帮助您讲述数据故事。

>[!VIDEO](https://video.tv.adobe.com/v/23994/?quality=12&learn=on)
