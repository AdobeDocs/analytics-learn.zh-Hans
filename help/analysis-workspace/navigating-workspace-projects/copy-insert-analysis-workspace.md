# 在Analysis Workspace复制和插入 {#copy-and-insert-in-analysis-workspace}

通过“复制+插入”，您可以将可视化从一个面板或项目复制到另一个面板或项目，还可以跨项目复制整个面板。

>[!VIDEO](https://video.tv.adobe.com/v/23230/?quality=12)

使用此功能将分析整理到逻辑数据故事和项目中，或从模板中复制来获得头部开始!
