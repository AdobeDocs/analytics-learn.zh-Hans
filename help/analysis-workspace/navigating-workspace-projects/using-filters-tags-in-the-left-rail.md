# 使用左侧边栏中的过滤器（标记）

过滤器（或标记）是一种快速、简便的方法，可用于在Analysis Workspace中过滤左边栏以找到所需的[!UICONTROL 组件]。 利用Adobe提供的现成标记或添加您自己的自定义标记来组织相关的[!UICONTROL 组件]。
