# 使用左侧边栏中的筛选条件（标记）

筛选条件（或标记）是一种在 Analysis Workspace 中对左侧边栏进行筛选以查找所需[!UICONTROL 组件]的快速轻松的方法。 利用 Adobe 提供的现成标记或添加您自己的自定义标记来整理相关[!UICONTROL 组件]。
