# 使用[!UICONTROL 同类群组表]进行[!UICONTROL 流失分析] {#churn-analysis-with-cohort-tables}

[!UICONTROL 同类群组表]提供两种分析类型 — [!UICONTROL 保留]和[!UICONTROL 流失]。 [!UICONTROL 保留]显示随着时间的推移保留的用户数，[!UICONTROL 流失]选项将描述丢失的用户数（1 — 保留）。
