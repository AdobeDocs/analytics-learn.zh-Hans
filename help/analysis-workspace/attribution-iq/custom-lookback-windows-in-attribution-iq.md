# Attribution IQ 中的自定义回看窗口期

自定义回顾窗口允许您将归因窗口扩展到报表范围以外（最长 90 天），并应用于报表范围内的每次转换。此选项通常会通过计算在上个月发生的交互，来提高在报告期初期发生的事件的归因准确性。
