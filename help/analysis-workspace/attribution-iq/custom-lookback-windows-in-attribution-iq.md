# Attribution IQ 中的自定义回顾时间范围

自定义回顾时间范围允许您将归因时间范围扩展到报告范围以外（最长 90 天），并应用于报告范围内的每次转化。 此选项通常会通过计算在之前月份发生的交互来提高在报告期初期所发生事件的归因准确性。
