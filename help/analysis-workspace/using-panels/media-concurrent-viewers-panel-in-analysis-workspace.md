# Analysis Workspace 中的“媒体并行查看者”面板

对于Media Analytics客户，通过“媒体并行查看者”面板，您可以了解出现并发峰值或发生锐减的位置，从而对内容质量和查看者参与度提供有价值的分析，并有助于对批量/规模进行疑难解答或规划。
