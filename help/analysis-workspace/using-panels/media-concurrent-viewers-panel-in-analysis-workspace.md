# Analysis Workspace 中的“媒体并行查看者”面板

对于Media Analytics客户，“媒体并发查看器”面板使您能够了解高峰期并发发生的位置或中断发生的位置，从而提供对内容质量和查看者参与情况的宝贵洞察，并帮助解决问题或规划批量/规模。
