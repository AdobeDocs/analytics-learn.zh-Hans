# Analysis Workspace 中的“媒体并行查看者”面板

对于 Media Analytics 客户，通过“媒体并行查看者”面板，您可以了解出现并发峰值或发生流失的位置，从而针对内容质量和查看者参与情况提供宝贵的洞察，并帮助进行故障排除或者容量/规模的规划。
