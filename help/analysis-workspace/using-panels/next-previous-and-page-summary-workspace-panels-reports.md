# 下一个/上一个及页面摘要工作区面板和报告

我们向Analysis Workspace中添加了两种新的面板类型 — “下一页”/“上一页”和“页面摘要”，并将它们列为现成报表的一部分。 这些功能使 Analysis Workspace 与一些更流行的 Reports &amp; Analytics 报告不相上下。
