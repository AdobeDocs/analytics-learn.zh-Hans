# 下一页/上一页和页面摘要工作区面板和报表

我们向Analysis Workspace中添加了两种新的面板类型 — “下一页”/“上一页”和“页面摘要”，并将它们列为现成报表的一部分。 这些功能使工作区与一些比较常用的Reports &amp; Analytics报表相同。
