# 下一个/上一个及页面摘要工作区面板和报告

我们已在 Analysis Workspace 中添加两种新面板类型，即下一个/上一个及页面摘要，并将其作为现成报告的一部分列出。 这些功能使 Analysis Workspace 与一些更流行的 Reports &amp; Analytics 报告不相上下。
