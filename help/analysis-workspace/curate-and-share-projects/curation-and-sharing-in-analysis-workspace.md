# Analysis Workspace 中的简化和共享

在 Analysis Workspace 中构建项目后，您可以[!UICONTROL 简化]项目并与组织中的其他团队和人员共享项目，以便他们能够探索项目并让数据为自己服务。本视频向您说明如何实施[!UICONTROL 简化]和共享。
