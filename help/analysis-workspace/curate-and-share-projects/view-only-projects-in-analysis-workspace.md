# Analysis Workspace 中的仅查看项目

可以将工作区项目仅以“可以查看”的形式共享给用户。当具有查看角色的收件人打开共享项目时，他们将会获得较为受限的项目体验，即，无左边栏而且只能进行有限的交互。
