# 在 Analysis Workspace 中实现项目共享

项目共享是一种通过 Analysis Workspace 向组织中的用户普及数据和见解的方法。您可以根据希望收件人获得的项目体验，为收件人提供以下三个项目角色之一：编辑、复制和查看。
