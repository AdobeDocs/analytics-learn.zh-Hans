# 以 CSV 格式下载 50000 项

通过以CSV格式下载项目选项，您可以为自由格式表中的某个维度下载最多50,000个项目，并应用区段和过滤器。 虽然 UI 在分页前将继续显示最多 400 个项目，但此选项允许您访问 Analysis Workspace 以外的更多数据行。
