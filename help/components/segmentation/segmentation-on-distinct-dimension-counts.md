# 对非重复维度计数分段 {#segmentation-on-distinct-dimension-counts}

通过[!UICONTROL 区段生成器]中新的[!UICONTROL 非重复计数]运算符，您可以根据任何[!UICONTROL 维度]中的唯一项目数进行分段。 过滤查看页面超过5次的访问，或购买产品超过5次的点击（即在单次订购中）。
