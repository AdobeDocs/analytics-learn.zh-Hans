# 在分段中使用“等于任意”运算符

区段生成器中已添加“等于任意”和“不等于任意”区段运算符。 使用这些运算符可筛选特定项目的列表，而不是使用多个OR语句或“包含任意”。 使用这些运算符而不是“包含任意”也有助于提高查询速度。
