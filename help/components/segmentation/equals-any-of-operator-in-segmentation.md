# 在分段中使用&quot;等于任意&quot;运算符

已将“等于任意”和“不等于任意”区段运算符添加到区段生成器中。 使用这些运算符可以过滤到特定项的列表，而不是使用多个OR语句或“包含任意项”。 使用这些运算符而不是“包含任何”也有助于提高查询速度。
