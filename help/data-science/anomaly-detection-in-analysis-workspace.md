# Analysis Workspace 中的[!UICONTROL 异常检测] {#anomaly-detection-in-analysis-workspace}

Analysis Workspace 自动检测任何时间序列[!UICONTROL 可视化图表]或数据表中的数据异常。 用户无需执行任何其他操作便可捕获“未知事件”！
